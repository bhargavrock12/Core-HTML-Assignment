HTML Tags


  1. <html>

    The HTML tag is the root element of an HTML document. It contains all other elements and defines the document type.

  2. <head>

    The head tag contains metadata about the document, such as the title, character encoding, stylesheets, and scripts.

  3. <body>

    The body tag contains the content of the HTML document, such as text, images, links, forms, tables, lists, etc.

  4. <p>

    The paragraph tag is used to define a paragraph of text within an HTML document. It is typically used to group a series of sentences together.

  5. <h1>

    The heading tag is used to define a heading within an HTML document. It can be used to define headings of different levels, from h1 (the most important) to h6 (the least important).

  6. <ol>

    The ordered list tag is used to define an ordered list within an HTML document. It is typically used to group a series of items together in a specific order.

  7. <li>

    The list item tag is used to define a list item within an ordered or unordered list within an HTML document.

  8. <ul>

    The unordered list tag is used to define an unordered list within an HTML document. It is typically used to group a series of items together in a list.

  9. <title>

    The title tag is used to define the title of an HTML document. It is typically displayed in the browser's title bar and is also used by search engines as the title of the page in their results.

  10. <a>

    The anchor tag is used to define a hyperlink within an HTML document. It is typically used to link to another web page, email address, or other resource.

  11. <img>

    The image tag is used to embed an image within an HTML document. It is typically used to display images on a web page.

  12. <div>

    The div tag is used to define a division or a section within an HTML document. It is typically used to group a series of elements together for styling purposes.

  13. <span>

    The span tag is used to define a section of text within an HTML document. It is typically used to group a series of elements together for styling purposes.

  14. <table>

    The table tag is used to define a table within an HTML document. It is typically used to display data in a tabular format.

  15. <tr>

    The table row tag is used to define a row within a table within an HTML document It is typically used to group a series of table data cells together.

  16. <td>

    The table data tag is used to define a cell within a table within an HTML document It is typically used to display data within a table.

  17. <form>

    The form tag is used to define a form within an HTML document. It is typically used to collect user input.

  18. <input>

    The input tag is used to define an input field within a form within an HTML document It is typically used to collect user input.

  19. <button>

    The button tag is used to define a button within a form within an HTML document. It is typically used to submit a form or perform an action.

  20. <label>

    The label tag is used to define a label for an input field within a form within an HTML document. It is typically used to provide a description of the input field.

  21. <br>

    The line break tag is used to insert a line break within an HTML document. It is typically used to separate lines of text.

  22. <hr>

    The horizontal rule tag is used to insert a horizontal rule within an HTML document. It is typically used to separate sections of content.

  23. <meta>

    The meta tag is used to define metadata within an HTML document. It is typically used to provide information about the document, such as its title, description, and keywords.

  24. <link>

    The link tag is used to define a link within an HTML document. It is typically used to link to an external resource, such as a stylesheet or an image.

  25. <script>

    The script tag is used to define a script within an HTML document. It is typically used to add interactivity to a web page.
